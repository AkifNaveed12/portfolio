# Muhammad Akif Naveed — Personal Portfolio

> Software Engineering × AI Engineering · COMSATS University Islamabad, Wah Campus

A premium dark-themed personal portfolio website built with pure HTML, CSS, and Vanilla JavaScript — no frameworks, no dependencies. Features animated particle backgrounds, typing effects, scroll-reveal animations, project showcases, research highlights, certifications, and achievements.

**Live:** [akif-naveed.vercel.app](https://akif-naveed.vercel.app) &nbsp;·&nbsp; **GitHub:** [AkifNaveed12](https://github.com/AkifNaveed12)

## Local Setup

No build step needed — just open the file:

```bash
git clone https://github.com/AkifNaveed12/portfolio.git
cd portfolio
# Open index.html in any browser
```
